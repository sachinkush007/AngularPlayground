Date Pipe Example:

{{ today | date:'medium' }}

Uppercase and Lowercase Pipe Example:

{{ 'hello world' | uppercase }}

{{ 'HELLO WORLD' | lowercase }}

Currency Pipe Example:

{{ price | currency:'USD':'symbol' }}

Decimal Pipe Example:

{{ pi | number:'1.2-2' }}

Percent Pipe Example:

{{ percentage | percent }}

Chaining pipe

{{ 'helloworld' | uppercase | slice:0:5 }}

{{ price | currency:'USD':'symbol' | percent }}
